# C Programming Exercises

This directory contains various C programs and mini-projects developed during my Computer Engineering studies at METU NCC.

The projects cover fundamental and intermediate topics in C programming including data structures, algorithms, file processing, and system simulation programs.

Many of the projects were developed for courses such as:

- **CNG 140 – C Programming**
- **CNG 213 – Data Structures**
- **CNG 315 – Algorithms**

---

# Repository Structure

The projects in this folder range from small programming exercises to larger systems demonstrating common computer science data structures.

## Data Structures Projects

These projects implement core data structures and algorithms.

### AVL Fishdom Analysis
AVL tree implementation used to analyze fish population data while maintaining balanced search performance.

### BST
Binary Search Tree based instructor record manager supporting insertion, search, and traversal operations.

### Student Book Borrowing Management
Hash table implementation with multiple collision resolution techniques (linear probing, quadratic probing, double hashing) used to manage student borrowing records.

### Social Network
Graph based social network simulation implementing DFS traversal to find connections between users.

### Transportation Company
Directed weighted graph representing a warehouse transportation network and route analysis.

### Eye Data Management

### Patient Records

### Lung Cancer Data

### Summer Internship Program

### Plane Ticket Program
